Gerenciamento de registros via planilha

Este sistema permite importar arquivos .xlsx e visualizar os registros em cards organizados por nome, cidade e ID. Basta carregar sua planilha e os dados aparecem automaticamente, com filtros em tempo real para facilitar a busca.

Não tem um arquivo? Sem problema — gere dados de exemplo com um clique e explore todas as funcionalidades do dashboard antes de importar sua própria planilha.

Como usar

  • Acesse o Dashboard pelo menu "Páginas" no canto superior direito
  • Clique em "Selecionar .xlsx" e escolha sua planilha com colunas id, nome e cidade
  • Os registros são carregados automaticamente em cards individuais
  • Use os filtros de Nome, Cidade ou ID para buscar em tempo real
  • Sem arquivo? Clique em "Dados de exemplo" para ver o sistema funcionando
